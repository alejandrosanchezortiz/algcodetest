Summary:

We have a number of users that we need to keep track of, we also wish to keep track of the connections between users.

Menu options:

Add user
- Opens a form to create new users
- Names must be unique
Connect Users
- Opens a form to connect users by clicking on the users available for connection
- Connections are done always both ways: user1-->user2 and user2-->user_1
View Connections
- Retrieve all the users that are directly connected to a given user
- Deleting users is also allowed within this report. Deleting an user will delete her/his connections as well
View Chart
- Opens a chart that shows the connections between the users
